Privacy Policy

Application: BrettOS Automation
Operator: BMore Management / Brett
Effective Date: July 17, 2026
Contact: brett@bmoremanagement.com

1. Overview

BrettOS Automation is a private, internal business tool used solely by its operator to automate accounting and bookkeeping workflows within QuickBooks Online. It is not a consumer-facing application and is not distributed to third parties.

2. Data Accessed

This application accesses QuickBooks Online data including invoices, bills, vendors, and payment records. All data belongs to the operator's own QuickBooks Online company file. No data is collected from or about third-party end users.

3. How Data Is Used

Data accessed via the QuickBooks Online API is used exclusively to automate internal accounting tasks such as creating invoices, recording bills, and syncing financial records. Data is not sold, shared, or transmitted to any third party.

4. Data Storage

API credentials (OAuth tokens) are stored securely as encrypted secrets in GitHub Actions. No QuickBooks data is stored persistently outside of QuickBooks Online itself.

5. Data Retention

OAuth refresh tokens are retained only as long as needed to maintain API access. Tokens are revocable at any time from the QuickBooks Online account settings.

6. Security

All API communication uses HTTPS/TLS. OAuth tokens are stored as encrypted environment secrets and are never exposed in logs or source code.

7. Contact

Questions regarding this policy may be directed to brett@bmoremanagement.com.
